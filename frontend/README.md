# Gurram Vasudha | Portfolio

A premium, high-end personal portfolio featuring a futuristic dark-mode design with neon accents and immersive animations. Built with React, Vite, and Tailwind CSS.

## Features

- **Modern UI/UX**: Futuristic dark theme with glassmorphism effects.
- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile).
- **Interactive Animations**: Smooth transitions using GSAP and Framer Motion.
- **Bento Box Layout**: Clean organization of skills and experience.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Animations**: Framer Motion, GSAP, Typed.js
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vasudha-vasu/gurram-vasudha-portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

To build the project for production:

```bash
npm run build
```

The output will be in the `dist` directory.
